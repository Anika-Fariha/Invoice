# Invoice
https://invoice-git-main-anika-farihas-projects.vercel.app/

A responsive web application for generating and managing invoices, built with Next.js and Tailwind CSS.

## Features

- Create, edit and view invoices  
- Responsive design
- Easy to customise and extend  

## Tech Stack

- **Frontend / Framework**: Next.js (JavaScript)  
- **Styling**: Tailwind CSS
